# Home

Welcome to the official wiki for Paper-Prefix!

Paper-Prefix is a simple, yet powerful prefix plugin for Paper servers. It allows players to set their own prefixes, which are displayed in chat and above their heads.

## Features

*   Set your own prefix with a command.
*   Supports color codes.
*   Prefixes are displayed in chat and nametags.
*   PlaceholderAPI support.
*   Easy to use and configure.
