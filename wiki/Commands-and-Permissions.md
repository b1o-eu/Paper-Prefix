# Commands and Permissions

This page documents all the commands and permissions available in Paper-Prefix.

## Commands

*   `/prefix set <text> [color]`: Sets your prefix.
    *   Example: `/prefix set Admin red`
*   `/prefix clear`: Clears your prefix.
*   `/prefix join <player>`: Copies another player's prefix.
*   `/prefix help`: Shows the help message.
*   `/status <text> [color]`: An alias for `/prefix set`.

## Permissions

*   `paperprefix.use`: Allows a player to use the `/prefix` and `/status` commands to set their own prefix.
*   `paperprefix.colors`: Allows a player to use color codes in their prefix.
*   `paperprefix.join`: Allows a player to use `/prefix join`.
*   `paperprefix.admin`: Grants access to all Paper-Prefix commands.
