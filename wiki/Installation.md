# Installation

1.  Download the latest version of Paper-Prefix from the [releases page](https://github.com/b1o-eu/Paper-Prefix/releases).
2.  Place the downloaded `.jar` file into your server's `plugins` folder.
3.  Restart your server.
4.  (Optional) Install [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi) to use placeholders.
